You can sort cards by Due Date, Title (Alphabetically), Votes, Last activity date, Date of appearence in the list, Creation Date and Labels.
The Apply current sorting menu item will save the current sorting in Trello, which means that current cards order will become the actual cards positions.

To access the sorting menu click the sort icon on the list header:  (put mouse over to enlarge)

Default Label's sorting priority:
(Drag the label to change the priority)

If you want to set a specific Labels sorting order for a particular board, click on the configure link in the Labels sort menu line:  

Important!
If you drag and drop the card to the sorted list the card's position in the list will not be preserved, it could appear anywhere in the list. So,it is recommended to reset the list sorting before moving the card to it. However, moving the cards from the sorted list is fine.

One more thing: After sorting is applied to the list, the list will be resorted every time any card is changed there, so if you see the sorted lists blinking sometimes - don't pay attention to it.

You can connect cards with each other forming parent-child and(or) siblings relationships.

Important: Connected cards urls are stored in the card's description at the very bottom under the "------" line, so please avoid manual editing of the text after the line.
For now, if a card with connections is deleted, you will need to manually remove references to the card in the connected card's description. This will be fixed soon.
(put mouse over to enlarge)

To access the connections menu click connection icon on the right side of the card title:

A connections menu will appear showing possible connecting options and the counts of existed connections in parenthesis:

You can choose either Show or Edit connections of the corresponding type.

Connections menu is also accessible from the card details window using the Connections button:

If the card has any connection(s), there will be a special icon(badge) in the badges area of the card:

To edit the connections click Edit in the Connections Menu and select the connected cards by simply clicking on them. After selection is done click Done link at the top of the page:

The cards count is shown in the title of the list

You can have badges in the header of the list showing the data aggregated from your custom fields.
For example an average or sum of all values of the custom field of the cards in the list.
